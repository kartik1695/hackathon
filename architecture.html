KEY DATA FLOWS
🔄

Request Flows

Leave Application Flow
POST /api/leaves/ → LeaveRequest Created → Django Signal → Celery: leave queue → AI Agent Graph → SPOF Check → Conflict Check → RAG Policy → LLM Decision → Update ai_context_card
AI Chat Flow
POST /api/ai/query/ → Get/Create ChatSession → ContextService → Intent Router → MCP Tools → RAG Retrieval → LLM Generate → Store + Return
Daily Burnout Scan (2 AM)
Celery Beat → burnout_scan_all → ai_heavy queue → burnout node → Redis Cache TTL=24h → Manager Dashboard
Policy RAG Ingestion
CLI: ingest_rag_docs → Chunkers (Simple/Meta/Schema) → Embedding Provider → pgvector Store → PolicyChunk rows
