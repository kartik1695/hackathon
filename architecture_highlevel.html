💬 Multi-turn Leave Collection

  • Parse "2 SL, 3 CL, 1 PL" naturally
  • Ask dates + reason per type
  • Hard block on policy violations
  • Context switch mid-flow (policy Q&A)
  • Batch apply on confirmation
Stages: collecting → policy check → blocked/confirm → applying → done
